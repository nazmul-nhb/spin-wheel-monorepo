1. await widget.spin()

The recommended way to collect results. widget.spin() returns a Promise<SpinResult> that resolves after the animation completes.

Result

Click "Spin!" to see the result here.

Source Code

import { SpinWheelWidget } from '@spin-wheel/widget';

const widget = SpinWheelWidget.create('#wheel', {
  segments: [
    { id: 'pizza', label: '🍕 Pizza', weight: 2 },
    { id: 'tacos', label: '🌮 Tacos', weight: 3 },
    // ...
  ],
});

// ✅ Await the promise — resolves after animation
const result = await widget.spin();

console.log(result.index);          // 0-based index
console.log(result.segment.label);  // "🌮 Tacos"
console.log(result.segment.id);     // "tacos"
console.log(result.segment.data);   // custom payload
console.log(result.finalAngle);     // rotation degrees
